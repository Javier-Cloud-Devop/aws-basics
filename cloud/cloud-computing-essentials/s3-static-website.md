# Static Website Hosting with Amazon S3

## Description
This lab consisted of configuraing an Amazon S3 bucket to host a static website using AWS Management Console (GUI)

## Steps performed
- Enabled static website hosting on an S3 bucket
- Configured index and error documents
- Renamed the index file to match the configured entry point
- Reviwed bucket permissions
- Allowed public read-only access to website files

## Key concepts learned
- How Amazon S3 can serve static website content
- The role of index and error documents
- Houw bucket policies and public access setting affects accesibility
