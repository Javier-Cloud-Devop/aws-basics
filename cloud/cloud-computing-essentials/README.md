# Cloud Computing Essentials

Hands-on labs completed as part of the AWS Skill Builder Cloud Computing Essential course. 

## Labs completed
- Static website hosting using Amazon S3
- EC2 deployment across multiple Availability Zones
- VPC networking concepts and security groups
- Cloud economics and EC2 cost estimation
- VPC peering and inter-VPC communication
- Amazon RDS bacis with Multi-AZ and read replicas
- Fisr NOSQL database using Amazon DynamoDB
- Auto-healing and scaling applications using EC2 AUto Scaling (shceduled scaling)
- Highly available web applications using ALB and Auto Scaling (multi-AZ)

## Tools used
- AWS Management Console (GUI)
